CompTIA Network+ (N10-009) Practice Questions

Domain 2.0: Network Implementations

Objective 2.1: Routing Technologies

Question 1

Which of the following routing technologies uses a successor and a feasible successor?

A. IS-IS

B. OSPF

C. BGP

D. EIGRP

Show Answer & Explanation ▶

Correct Answer: D. EIGRP

Why this is correct: EIGRP (Enhanced Interior Gateway Routing Protocol) is well-known for its fast convergence time, which it achieves by pre-calculating backup routes. The primary, best path to a destination is called the **Successor**. A pre-qualified, loop-free backup path is called the **Feasible Successor**. If the Successor route fails, the router can immediately switch to the Feasible Successor without needing to recalculate the entire network topology, making it very efficient.

Why others are incorrect:

  • OSPF and IS-IS: These protocols build a map of the network and run an algorithm to find the single best path. If that path fails, they must run the algorithm again to find a new best path. They don't use the "feasible successor" terminology or concept.
  • BGP: As the routing protocol for the internet, BGP uses a complex set of path attributes (like AS-Path, Local Preference) to determine the best path between different autonomous systems. It does not use the successor/feasible successor model.

Question 2

A network administrator wants to configure a backup route in case the primary route fails. A dynamic routing protocol is not installed on the router. Which of the following routing features should the administrator choose to accomplish this task?

A. Neighbor adjacency

B. Link state flooding

C. Administrative distance

D. Hop count

Show Answer & Explanation ▶

Correct Answer: C. Administrative distance

Why this is correct: Administrative Distance (AD) is a number (0-255) that a router uses to rank the trustworthiness of a routing source; a lower number is better. Since no dynamic protocol is used, the admin must use static routes. They can configure a "floating static route" by creating two static routes: one with the default AD (e.g., 1) and a backup route with a higher AD (e.g., 5). The router will install the primary route. If that route's interface goes down, the router removes it from the routing table and "floats" the backup route with the higher AD into place.

Why others are incorrect:

  • Neighbor adjacency & Link state flooding: These are functions of dynamic routing protocols like OSPF and EIGRP, which are not in use here.
  • Hop count: This is a metric used by protocols like RIP to measure the number of routers a packet must cross. It doesn't determine the trustworthiness or selection between two static routes.

Question 3

Which of the following protocols is used to route traffic on the public internet?

A. BGP

B. OSPF

C. EIGRP

D. RIP

Show Answer & Explanation ▶

Correct Answer: A. BGP

Why this is correct: The internet is a massive network of networks, where each network is an "Autonomous System" (AS). **BGP (Border Gateway Protocol)** is the only protocol designed to exchange routing information between these different ASes. It is the standardized Exterior Gateway Protocol (EGP) that makes global internet routing possible.

Why others are incorrect:

  • OSPF, EIGRP, and RIP: These are Interior Gateway Protocols (IGPs). They are designed to manage routing *within* a single autonomous system (like a single company's corporate network), not between them on the public internet.

Question 4

Which of the following routing protocols uses an autonomous system number?

A. IS-IS

B. OSPF

C. BGP

D. EIGRP

Show Answer & Explanation ▶

Correct Answer: C. BGP

Why this is correct: An **Autonomous System (AS)** is a collection of routers under a single administrative authority, identified by a unique **Autonomous System Number (ASN)**. **BGP** is the protocol that routes traffic *between* these different ASes on the internet. It fundamentally relies on ASNs to identify networks and make routing decisions.

Why others are incorrect:

  • EIGRP: While EIGRP configuration requires an "autonomous system" number, it's used locally to identify which routers belong to the same EIGRP domain and is not a globally unique number used for internet routing.
  • OSPF and IS-IS: These protocols operate within a single AS and do not use an ASN as part of their core routing logic. They use concepts like areas and process IDs for organization.

Question 5

Which of the following protocols has a default administrative distance value of 90?

A. RIP

B. EIGRP

C. OSPF

D. BGP

Show Answer & Explanation ▶

Correct Answer: B. EIGRP

Why this is correct: Each routing protocol is assigned a default **Administrative Distance (AD)** to indicate its trustworthiness (lower is better). For Cisco routers, the default AD for internal **EIGRP** routes is **90**. This makes it more preferred than OSPF (110) and RIP (120) but less preferred than a directly connected interface (0) or a static route (1).

Why others are incorrect (Common Cisco Defaults):

  • RIP: Default AD is 120.
  • OSPF: Default AD is 110.
  • BGP: Default AD is 20 for external BGP (eBGP) and 200 for internal BGP (iBGP).

Question 6

In an environment with one router, which of the following will allow a network engineer to communicate between VLANs without purchasing additional hardware?

A. Subinterfaces

B. VXLAN

C. Layer 3 switch

D. VIP

Show Answer & Explanation ▶

Correct Answer: A. Subinterfaces

Why this is correct: This scenario describes a classic "router-on-a-stick" configuration. **Subinterfaces** are logical, virtual interfaces created on a single physical router port. Each subinterface can be assigned to a different VLAN and given its own IP address to act as the default gateway for that VLAN. This allows a single physical router connection to route traffic between multiple VLANs.

Why others are incorrect:

  • VXLAN: This is an advanced tunneling protocol used to create virtual networks, often in large data centers. It's overly complex for this simple task.
  • Layer 3 switch: This is additional hardware that can perform routing, but the question specifies not purchasing additional hardware.
  • VIP (Virtual IP): This is a single IP address shared among multiple devices, typically used for redundancy (like in FHRP) or load balancing, not for inter-VLAN routing itself.

Question 7

Which of the following allows an organization to map multiple internal devices to a single external-facing IP address?

A. NAT

B. BGP

C. OSPF

D. FHRP

Show Answer & Explanation ▶

Correct Answer: A. NAT

Why this is correct: NAT (Network Address Translation) is a technology used on routers and firewalls to translate private, internal IP addresses (like 192.168.1.100) into one or more public, external IP addresses. This allows many devices on a private network to share a single public IP address to access the internet, conserving the limited supply of public IPv4 addresses.

Why others are incorrect:

  • BGP and OSPF: These are routing protocols that determine the best path for data, they don't translate addresses.
  • FHRP (First Hop Redundancy Protocol): This is a group of protocols (like HSRP, VRRP) that provide a redundant default gateway; it doesn't map multiple devices to one IP.

Question 8

Which of the following should a company implement in order to share a single IP address among all the employees in the office?

A. STP

B. BGP

C. PAT

D. VXLAN

Show Answer & Explanation ▶

Correct Answer: C. PAT

Why this is correct: PAT (Port Address Translation), also known as NAT Overload, is the most common form of NAT. It maps multiple private IP addresses to a single public IP address by also translating the source port number. For example, PC1's traffic might leave as `PublicIP:3001` and PC2's as `PublicIP:3002`. The router keeps a table of these mappings to direct return traffic to the correct internal device. This allows thousands of employees to share one public IP address.

Why others are incorrect:

  • STP (Spanning Tree Protocol): Prevents loops on Layer 2 switches.
  • BGP (Border Gateway Protocol): An internet routing protocol.
  • VXLAN (Virtual Extensible LAN): A network virtualization technology.

Question 9

Which of the following technologies uses a VIP to provide gateway redundancy between two routers?

A. LACP

B. PAT

C. FHRP

D. BGP

Show Answer & Explanation ▶

Correct Answer: C. FHRP

Why this is correct: **FHRP (First Hop Redundancy Protocol)** is a category of protocols (like HSRP, VRRP, GLBP) designed to prevent the default gateway from being a single point of failure. Two or more routers work together to present a single **VIP (Virtual IP)** address to the network clients. One router is "active" and handles the traffic, while the other(s) are on standby. If the active router fails, a standby router instantly takes over the VIP, ensuring clients always have a working gateway.

Why others are incorrect:

  • LACP (Link Aggregation Control Protocol): Bundles multiple physical ports into a single logical link for increased bandwidth and redundancy, but doesn't provide gateway redundancy.
  • PAT (Port Address Translation): Translates private IPs to a public IP.
  • BGP (Border Gateway Protocol): An internet routing protocol.

Question 10

A systems administrator installs multiple web servers and intends to use the First Hop Redundancy Protocol to control routing to the servers. Which of the following types of IP addresses should the administrator use to support this routing approach?

A. Virtual

B. Gateway

C. Static

D. Dynamic

Show Answer & Explanation ▶

Correct Answer: A. Virtual

Why this is correct: As explained in the previous question, **First Hop Redundancy Protocols (FHRP)** work by creating a **Virtual IP (VIP)** address. This VIP is not tied to any single physical device but is shared among the redundant routers or servers. Clients are configured to use this VIP as their destination, and the FHRP protocol ensures that one of the active devices will always respond to traffic sent to that virtual address.

Why others are incorrect:

  • Gateway, Static, Dynamic: These terms describe roles or assignment methods for IP addresses, but the specific type of address used by FHRP to create a redundant, shared presence is a virtual IP.

Question 11

Which of the following most likely requires the use of subinterfaces?

A. A router with only one available LAN port

B. A firewall performing deep packet inspection

C. A hub utilizing jumbo frames

D. A switch using Spanning Tree Protocol

Show Answer & Explanation ▶

Correct Answer: A. A router with only one available LAN port

Why this is correct: The primary use case for **subinterfaces** is to allow a single physical router interface to handle traffic for multiple VLANs. This is called a "router-on-a-stick" configuration. If a router needs to provide routing for several networks (VLANs) but only has one physical port to connect to the switch, subinterfaces are the solution. The physical port is connected to a trunk port on the switch, and each subinterface is configured to handle traffic for a specific VLAN tag.

Why others are incorrect:

  • Firewall, Hub, Switch using STP: None of these technologies inherently require subinterfaces. They are all related to different networking functions (security, Layer 1 connectivity, loop prevention).

Objective 2.2: Switching Technologies

Question 12

A network administrator is configuring a new switch and wants to connect two ports to the core switch to ensure redundancy. Which of the following configurations would meet this requirement?

A. Full duplex

B. 802.1Q tagging

C. Native VLAN

D. Link aggregation

Show Answer & Explanation ▶

Correct Answer: D. Link aggregation

Why this is correct: Link Aggregation** (often using the protocol LACP, 802.3ad) is a technique that bundles multiple physical network connections into a single logical link. This provides two key benefits: increased total bandwidth (the sum of the links) and **redundancy**. If one of the physical links in the bundle fails, traffic will automatically continue to flow over the remaining active links, preventing a network outage.

Why others are incorrect:

  • Full duplex: Allows data to be sent and received simultaneously on a single link, but doesn't provide redundancy if the link itself fails.
  • 802.1Q tagging & Native VLAN: These are features used to manage VLANs on trunk links, they don't provide physical link redundancy.

Question 13

A network administrator received complaints of intermittent network connectivity issues. The administrator investigates and finds that the network design contains potential loop scenarios. Which of the following should the administrator do?

A. Enable spanning tree.

B. Configure port security.

C. Change switch port speed limits.

D. Enforce 802.1Q tagging.

Show Answer & Explanation ▶

Correct Answer: A. Enable spanning tree.

Why this is correct: A switching loop occurs when there are multiple paths between two switches, causing broadcast frames to circulate endlessly, consuming all available bandwidth and crashing the network. The **Spanning Tree Protocol (STP)** is specifically designed to prevent this. It intelligently detects redundant links and blocks traffic on them, ensuring there is only one active path at any time. If the primary path fails, STP automatically unblocks the redundant path to maintain connectivity.

Why others are incorrect:

  • Port security: Limits which MAC addresses can connect to a port; it doesn't prevent loops.
  • Port speed limits: Manages bandwidth but doesn't solve the core issue of a broadcast storm from a loop.
  • 802.1Q tagging: Used for VLANs, not loop prevention.

Question 14

A VoIP phone is plugged in to a port but cannot receive calls. Which of the following needs to be done on the port to address the issue?

A. Trunk all VLANs on the port.

B. Configure the native VLAN.

C. Tag the traffic to voice VLAN.

D. Disable VLANs.

Show Answer & Explanation ▶

Correct Answer: C. Tag the traffic to voice VLAN.

Why this is correct: Modern networks separate voice and data traffic into different VLANs for performance and security. A VoIP phone needs to send its traffic with a specific VLAN tag so the switch knows to place it on the **voice VLAN**. This is typically done by configuring a "voice VLAN" on the switch port, which tells the switch to accept tagged voice traffic from the phone while still allowing untagged data traffic from a PC plugged into the phone.

Why others are incorrect:

  • Trunk all VLANs: This is unnecessary and a security risk. The port only needs access to the data and voice VLANs.
  • Configure the native VLAN: The native VLAN is for untagged traffic, which is usually data, not voice.
  • Disable VLANs: This would merge all traffic onto one network, defeating the purpose of separating voice and data.

Question 15

A network administrator is deploying a new switch and wants to make sure that the default priority value was set for a spanning tree. Which of the following values would the network administrator expect to see?

A. 4096

B. 8192

C. 32768

D. 36684

Show Answer & Explanation ▶

Correct Answer: C. 32768

Why this is correct: The Spanning Tree Protocol (STP) elects a "root bridge" for the network based on the Bridge ID, which is a combination of a priority value and the switch's MAC address. The switch with the lowest Bridge ID wins. The priority value can be set in increments of 4096. By default, all switches are configured with a priority value of **32768**.

Why others are incorrect:

  • 4096 and 8192: These are valid priority values, but they are not the default. An administrator would set these lower values to influence which switch becomes the root bridge.
  • 36684: This is not a valid STP priority value, as they must be in multiples of 4096.

Question 16

A network administrator is expanding a network and wants to ensure no unauthorized redundant links are present. Which of the following should the administrator use to identify and block redundant links?

A. SDN

B. STP

C. LLDP

D. OSPF

Show Answer & Explanation ▶

Correct Answer: B. STP

Why this is correct: This is the core function of **STP (Spanning Tree Protocol)**. It is designed specifically to operate on Layer 2 switched networks to prevent broadcast storms by detecting and blocking redundant paths. It creates a single, loop-free logical topology from a physical topology that may contain loops for redundancy.

Why others are incorrect:

  • SDN (Software-Defined Networking): A network architecture, not a specific loop-prevention protocol.
  • LLDP (Link Layer Discovery Protocol): Used by devices to advertise their identity and capabilities to neighbors; it discovers links but does not block them.
  • OSPF (Open Shortest Path First): A Layer 3 routing protocol that prevents routing loops, but not Layer 2 switching loops.

Question 17

A network manager connects two switches together and uses two connecting links. Which of the following configurations will prevent Layer 2 loops?

A. 802.1Q tagging

B. Full duplex

C. Link aggregation

D. QoS

Show Answer & Explanation ▶

Correct Answer: C. Link aggregation

Why this is correct: While STP also prevents loops, **Link Aggregation (802.3ad)** is another way to handle this specific scenario. It bundles the two physical links into a single logical channel. From the perspective of Spanning Tree, it's just one link, so no loop is formed. This is often a better solution than STP alone because it allows you to use the bandwidth of both links simultaneously, whereas STP would block one of the links.

Why others are incorrect:

  • 802.1Q tagging: Manages VLANs, does not prevent loops.
  • Full duplex: A port setting, does not prevent loops.
  • QoS (Quality of Service): Prioritizes traffic, does not prevent loops.

Question 18

A network administrator deploys several new desk phones and workstation cubicles. Each cubicle has one assigned switchport. The administrator run the following commands: `switchport mode access` `switchport voice vlan 69`. With which of the following VLANs will the workstation traffic be tagged?

A. Private VLAN

B. Voice VLAN

C. Native VLAN

D. Data VLAN

Show Answer & Explanation ▶

Correct Answer: D. Data VLAN

Why this is correct: The command `switchport mode access` assigns the port to a single VLAN, which is the **Data VLAN** (also called the access VLAN). The command `switchport voice vlan 69` adds a second, special VLAN for tagged voice traffic. The workstation connected to the phone is a standard data device; it does not tag its frames. Therefore, its traffic will be handled by the port's primary access VLAN, the Data VLAN. The phone itself will tag its voice packets for VLAN 69.

Why others are incorrect:

  • Private VLAN: A specific security feature, not configured here.
  • Voice VLAN: This is where the phone's traffic goes, not the workstation's.
  • Native VLAN: This term applies to trunk ports, not access ports.

Question 19

A network engineer is deploying switches at a new remote office. The switches have been preconfigured with hostnames and STP priority values. Based on the following table... Which of the following switches will become the root bridge?

A. core-sw01

B. access-sw01

C. distribution-sw01

D. access-sw02

Show Answer & Explanation ▶

Correct Answer: A. core-sw01

Explanation: The Spanning Tree Protocol (STP) elects a root bridge based on the lowest Bridge ID. The Bridge ID is composed of a priority value and the switch's MAC address. Since the table is missing, we must rely on network design best practices. The **core switch** is the center of the network and should be the most stable and powerful. To ensure a predictable and efficient traffic path, administrators intentionally configure the core switch to have the lowest STP priority, forcing it to become the root bridge. Therefore, `core-sw01` is the most logical choice.

Question 20

A storage network requires reduced overhead and increased efficiency for the amount of data being sent. Which of the following should an engineer most likely configure to meet these requirements?

A. Link speed

B. Jumbo frames

C. QoS

D. 802.1q tagging

Show Answer & Explanation ▶

Correct Answer: B. Jumbo frames

Why this is correct: Standard Ethernet frames have a maximum size (MTU) of 1500 bytes. For every frame sent, there is processing overhead for the headers. **Jumbo frames** increase the MTU to 9000 bytes or more. By sending much larger frames, you can transmit the same amount of data with fewer total frames. This reduces the overall header overhead and the CPU cycles needed to process the frames, leading to increased efficiency, which is especially beneficial for high-throughput storage networks.

Why others are incorrect:

  • Link speed: Increasing speed helps, but doesn't reduce the *overhead* per byte.
  • QoS: Prioritizes traffic but doesn't make the transmission itself more efficient.
  • 802.1Q tagging: Adds a 4-byte header to each frame, slightly *increasing* overhead.

Question 21

Which of the following can support a jumbo frame?

A. Access point

B. Bridge

C. Hub

D. Switch

Show Answer & Explanation ▶

Correct Answer: D. Switch

Why this is correct: Jumbo frames are a feature of modern Ethernet networks. To use them, every device in the data path—including the sending computer's NIC, the receiving computer's NIC, and all intermediary **switches**—must be configured to support the larger frame size. Modern managed switches are the key network infrastructure component that can be configured for jumbo frames.

Why others are incorrect:

  • Access point: Wi-Fi has its own frame size standards and does not typically support Ethernet jumbo frames.
  • Bridge & Hub: These are legacy devices. Hubs are Layer 1 and don't process frames, while older bridges would not support non-standard frame sizes.

Objective 2.3: Wireless Devices and Technologies

Question 22

A network administrator is performing a refresh of a wireless environment. As the APs are being placed, they overlap a little bit with each other. Which of the following 2.4GHz channels should be selected to ensure that they do not conflict?

A. 1, 3, 5

B. 1, 6, 11

C. 2, 6, 10

D. 3, 6, 9

Show Answer & Explanation ▶

Correct Answer: B. 1, 6, 11

Why this is correct: The 2.4GHz Wi-Fi band is divided into 11 channels (in North America), but each channel is 22MHz wide and overlaps with its neighbors. To avoid co-channel interference, you must select channels that are far enough apart. The only set of channels in the 2.4GHz band that do not overlap at all are **1, 6, and 11**. This is the standard best practice for deploying multiple access points in this frequency band.

Why others are incorrect:

  • All other options: In all other combinations listed, the channels are too close together and their signals would overlap, causing interference and poor performance. For example, channel 3 overlaps with channel 1, and channel 5 overlaps with channel 6.

Question 23

A company is implementing a wireless solution in a high-density environment. Which of the following 802.11 standards is used when a company is concerned about device saturation and coverage?

A. 802.11ac

B. 802.11ax

C. 802.11g

D. 802.11n

Show Answer & Explanation ▶

Correct Answer: B. 802.11ax

Why this is correct: 802.11ax (Wi-Fi 6)** was specifically designed to improve performance in high-density environments like stadiums, airports, and crowded offices. Its key feature is **OFDMA (Orthogonal Frequency-Division Multiple Access)**, which allows an access point to communicate with multiple devices simultaneously by dividing a channel into smaller sub-channels. This dramatically reduces contention and improves efficiency when many devices are competing for airtime.

Why others are incorrect:

  • 802.11ac (Wi-Fi 5): Introduced higher speeds but was less focused on efficiency in dense environments.
  • 802.11g and 802.11n (Wi-Fi 3 & 4): These are older standards that struggle significantly with device saturation.

Question 24

A network administrator is configuring a wireless network with an ESSID. Which of the following is a user benefit of ESSID compared to SSID?

A. Stronger wireless connection

B. Roaming between access points

C. Advanced security

D. Increased throughput

Show Answer & Explanation ▶

Correct Answer: B. Roaming between access points

Why this is correct: An SSID (Service Set Identifier) is the name of a single wireless network from one access point. An **ESSID (Extended Service Set Identifier)** is used when multiple access points are configured with the *same* network name and security settings. This allows a user's device to seamlessly **roam** from one access point's coverage area to another without losing their connection. The device sees it as one large, continuous network.

Why others are incorrect:

  • Connection strength, security, and throughput: These are determined by other factors like the Wi-Fi standard, distance from the AP, and security protocol used, not by the use of an ESSID itself.

Question 25

A wireless technician wants to implement a technology that will allow user devices to automatically navigate to the best available frequency standard. Which of the following technologies should the technician use?

A. Band steering

B. Wireless LAN controller

C. Directional antenna

D. Autonomous access point

Show Answer & Explanation ▶

Correct Answer: A. Band steering

Why this is correct: **Band steering** is a feature on dual-band (2.4GHz and 5GHz) access points. It actively encourages dual-band capable clients to connect to the less-congested and higher-performance 5GHz band, leaving the 2.4GHz band for older devices. The AP does this by making the 5GHz band look more attractive to the client device, effectively "steering" it to the better frequency.

Why others are incorrect:

  • Wireless LAN controller: Manages APs but is not the technology that steers clients.
  • Directional antenna: Focuses the RF signal in a specific direction but doesn't manage frequency selection.
  • Autonomous access point: A type of AP, but doesn't describe the specific feature.

Question 26

A network administrator upgraded the wireless access points and wants to implement a configuration that will give users higher speed and less channel overlap based on device compatibility. Which of the following will accomplish this goal?

A. 802.1X

B. MIMO

C. ESSID

D. Band steering

Show Answer & Explanation ▶

Correct Answer: D. Band steering

Why this is correct: This question is asking for the same concept as the previous one. **Band steering** accomplishes the goal by moving compatible devices to the 5GHz band. The 5GHz band offers higher speeds and has many more non-overlapping channels compared to the crowded 2.4GHz band. This results in a better experience for users with capable devices.

Why others are incorrect:

  • MIMO (Multiple-Input Multiple-Output): Uses multiple antennas to increase throughput, but doesn't address channel overlap.

Question 27

A company is building a new campus that will have wired and wireless devices. The network will be designed to support casual internet usage for guests and break rooms for employees. Which of the following authentication methods is best suited for guests given the requirements of this scenario?

A. ESP

B. PSK

C. WPA

D. IKE

Show Answer & Explanation ▶

Correct Answer: B. PSK

Why this is correct: **PSK (Pre-Shared Key)** is the authentication method used in WPA/WPA2/WPA3-Personal mode. It involves a single password (passphrase) that is shared with all authorized users. This is ideal for guest networks or small office environments because it's simple to manage—you just give the guest the password. For "casual internet usage," the complexity of enterprise authentication is not needed.

Why others are incorrect:

  • ESP (Encapsulating Security Payload) and IKE (Internet Key Exchange): These are protocols used within the IPsec suite to create VPNs, not for Wi-Fi authentication.
  • WPA (Wi-Fi Protected Access): This is the name of the security protocol suite itself, not the specific authentication method. PSK is a method *within* WPA.

Question 28

The network engineering team needs to implement a wireless network within two separate buildings. Once the network is set up, users will need to authenticate using RADIUS in order to access internal resources. Which of the following wireless technologies should the team implement?

A. Mesh

B. Enterprise

C. Ad hoc

D. Point to point

Show Answer & Explanation ▶

Correct Answer: B. Enterprise

Why this is correct: The key requirement here is authenticating with a **RADIUS** server. This is the defining feature of **WPA-Enterprise** mode (also known as 802.1X). In this mode, users don't share a single password. Instead, each user authenticates with their own unique credentials (username/password or certificate), which are verified by a central RADIUS server. This provides much stronger security and individual accountability, which is necessary for accessing internal corporate resources.

Why others are incorrect:

  • Mesh, Ad hoc, Point-to-point: These are network topologies or modes, not authentication methods. While an enterprise network might use a mesh topology, the authentication scheme itself is "Enterprise."

Question 29

A network administrator is configuring a wireless connection between two warehouses. Which of the following wireless network types would best accomplish this task?

A. Ad hoc

B. Infrastructure

C. Point-to-point

D. Mesh

Show Answer & Explanation ▶

Correct Answer: C. Point-to-point

Why this is correct: A **Point-to-Point (P2P)** wireless link is specifically designed to connect two fixed locations, like two buildings. It typically uses directional antennas on both ends to create a focused, high-speed wireless bridge, effectively replacing a physical cable run between the warehouses.

Why others are incorrect:

  • Ad hoc: A temporary, peer-to-peer network between client devices, not suitable for a permanent infrastructure link.
  • Infrastructure: The standard mode where clients connect to an access point to access a network. This describes the network *inside* the warehouse, not the link *between* them.
  • Mesh: Used to provide widespread coverage within an area using multiple interconnected APs, which is overly complex for connecting just two fixed points.

Question 30

A network engineer needs to deploy an access point at a remote office so that it will not communicate back to the wireless LAN controller. Which of the following deployment methods must the engineer use to accomplish this task?

A. ESSID

B. Lightweight

C. Autonomous

D. Ad hoc

Show Answer & Explanation ▶

Correct Answer: C. Autonomous

Why this is correct: An **Autonomous** access point (also called a "fat" or "standalone" AP) is a self-contained device. It has all the intelligence, configuration, and management software built-in. It operates independently and does not require a Wireless LAN Controller (WLC) to function. This is the correct choice for a site where a connection back to a central controller is not desired or possible.

Why others are incorrect:

  • Lightweight: A lightweight AP ("thin" AP) is the opposite; it has minimal intelligence and *must* communicate with a WLC for its configuration and operation.
  • ESSID: The name of the wireless network, not a deployment method.
  • Ad hoc: A mode of communication between client devices, not an AP deployment method.

Objective 2.4: Physical Installations

Question 31

Which of the following network cables involves bouncing light off of protective cladding?

A. Twinaxial

B. Coaxial

C. Single-mode

D. Multimode

Show Answer & Explanation ▶

Correct Answer: D. Multimode

Why this is correct: **Multimode** fiber optic cable has a relatively large core that allows multiple modes (paths) of light to travel down the cable simultaneously. These light rays travel by bouncing off the boundary of the core and the surrounding cladding at different angles. This design is suitable for shorter distances within a building.

Why others are incorrect:

  • Single-mode: Has a very small core and sends a single, straight beam of light directly down the center, allowing for much longer distances.
  • Twinaxial and Coaxial: These are copper cables that transmit electrical signals, not light.

Question 32

A network technician needs to install patch cords from the UTP patch panel to the access switch for a newly occupied set of offices. The patch panel is not labeled for easy jack identification. Which of the following tools provides the easiest way to identify the appropriate patch panel port?

A. Toner

B. Laptop

C. Cable tester

D. Visual fault locator

Show Answer & Explanation ▶

Correct Answer: A. Toner

Why this is correct: A **toner probe and tone generator** is the perfect tool for this job. The technician connects the tone generator to a wall jack in an office, which sends an audible tone down the copper cable. The technician then uses the wand-like probe at the patch panel, sweeping it across the ports until it beeps loudly, indicating it has found the other end of the cable. This allows for quick identification of unlabeled cables.

Why others are incorrect:

  • Laptop: Could be used to see if a link light comes up, but this is slow and inefficient for many ports.
  • Cable tester: Verifies the wiring of a cable but doesn't help locate it.
  • Visual fault locator: Used to find breaks in fiber optic cables by shining a visible red laser, not for copper UTP.

Question 34

A network administrator is in the process of installing 35 POE security cameras. After the administrator installed and tested the new cables, the administrator installed the cameras. However, a small number of the cameras do not work. Which of the following is the most likely reason?

A. Incorrect wiring standard

B. Power budget exceeded

C. Signal attenuation

D. Wrong voltage

Show Answer & Explanation ▶

Correct Answer: B. Power budget exceeded

Why this is correct: A Power over Ethernet (PoE) switch has a total **power budget**, which is the maximum amount of wattage it can supply across all of its ports combined. Each camera draws a certain amount of power. It is common that after connecting a certain number of devices, the switch's total power budget is reached. Any cameras plugged in after that point will not receive power and will fail to turn on, even if the data connection is fine.

Why others are incorrect:

  • Incorrect wiring standard: The admin tested the cables, so this is unlikely.
  • Signal attenuation: This would cause data loss or connection issues, but usually wouldn't prevent the camera from powering on entirely unless the cable run is extremely long.
  • Wrong voltage: PoE standards (like 802.3af/at/bt) involve a negotiation process to ensure the correct voltage is supplied. It's rare for a switch to send the "wrong" voltage.

Question 35

Which of the following is a location where a plenum cable is recommended?

A. Frequently traveled areas

B. Outdoor environments

C. Inside conduits

D. Airspaces above a ceiling

Show Answer & Explanation ▶

Correct Answer: D. Airspaces above a ceiling

Why this is correct: A **plenum** is the space used for air circulation in heating and air conditioning systems, which is often the space above a drop ceiling or under a raised floor. Fire safety codes are very strict about these areas. **Plenum-rated cable** has a special fire-retardant jacket (usually made of Teflon) that burns at a higher temperature and emits less toxic smoke than standard PVC jackets. It is required by law in most commercial buildings for any cables run through plenum spaces.

Why others are incorrect:

  • Frequently traveled areas, Outdoor environments, Inside conduits: These areas may require other types of cable (e.g., riser-rated, direct burial, armored), but plenum is specifically for air handling spaces.

Question 36

A technician is planning an equipment installation into a rack in a data center that practices hot aisle/cold aisle ventilation. Which of the following directions should the equipment exhaust face when installed in the rack?

A. Sides

B. Top

C. Front

D. Rear

Show Answer & Explanation ▶

Correct Answer: D. Rear

Why this is correct: The hot aisle/cold aisle design is the standard for data center cooling efficiency. Racks are arranged in rows where the fronts of the racks face each other (the cold aisle) and the backs of the racks face each other (the hot aisle). Cool air is pumped into the cold aisle. Equipment is installed to pull this cool air in through the **front** and expel hot exhaust out the **rear** into the hot aisle, where it is then returned to the air conditioning units. This prevents hot and cold air from mixing.

Why others are incorrect:

  • Sides, Top, Front: Directing exhaust in any of these directions would disrupt the hot/cold aisle airflow, reduce cooling efficiency, and cause equipment to overheat.

Question 37

Which of the following can protect network equipment from voltage variations?

A. MDF

B. Power strips

C. PDU

D. UPS

Show Answer & Explanation ▶

Correct Answer: D. UPS

Why this is correct: A **UPS (Uninterruptible Power Supply)** does more than just provide battery backup. It also contains circuitry for power conditioning. It can protect against voltage variations like sags (brownouts), surges, and spikes by regulating the incoming power and providing a clean, consistent voltage to the connected equipment. This is critical for protecting sensitive electronics.

Why others are incorrect:

  • MDF (Main Distribution Frame): A location for network cabling, not a power device.
  • Power strips: Simply provide more outlets; they offer no voltage protection.
  • PDU (Power Distribution Unit): A heavy-duty power strip for racks, which may have surge protection but typically lacks the advanced voltage regulation and battery backup of a UPS.

Question 38

Which of the following is the most cost-effective way to safely expand outlet capacity in an IDF?

A. PDU

B. Surge protector

C. UPS

D. Power strip

Show Answer & Explanation ▶

Correct Answer: A. PDU

Why this is correct: An IDF (Intermediate Distribution Frame) is a network closet that houses equipment in racks. A **PDU (Power Distribution Unit)** is essentially a professional, rack-mountable power strip designed for this environment. Question 39

Which of the following are environmental factors that should be considered when installing equipment in a building? (Choose two.)

A. Fire suppression system

B. UPS location

C. Humidity control

D. Power load

E. Floor construction type

F. Proximity to nearest MDF

Show Answer & Explanation ▶

Correct Answers: C. Humidity control and D. Power load

Why these are correct: Environmental factors relate to the physical conditions of the surrounding area that can affect equipment operation.

  • Humidity control: Too much humidity can cause condensation and corrosion on electronic components. Too little humidity increases the risk of electrostatic discharge (ESD), which can damage sensitive chips. Proper humidity control is critical.
  • Power load: This is the total amount of electrical power the equipment will draw. It's an essential environmental factor to consider to ensure the building's circuits can safely handle the demand without overloading and causing power failures or fires.

Why others are incorrect:

  • Fire suppression system & UPS location: These are crucial safety and infrastructure planning items, but they are systems you install *into* the environment, not properties *of* the environment itself like temperature or humidity.
  • Floor construction type: This relates to the physical support and weight capacity, a structural concern, not an environmental one.
  • Proximity to nearest MDF: This is a logistical and network design consideration related to cabling distance, not an environmental factor.
