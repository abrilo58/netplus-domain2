Objective 2.1: Routing Technologies

Question 1

[cite_start]

Which of the following routing technologies uses a successor and a feasible successor? [cite: 2]

A. [cite_start]IS-IS [cite: 3]

B. [cite_start]OSPF [cite: 5]

C. [cite_start]BGP [cite: 6]

D. [cite_start]EIGRP [cite: 7]

Show Answer & Explanation ▶

Correct Answer: D. EIGRP

Why this is correct: EIGRP (Enhanced Interior Gateway Routing Protocol) is well-known for its fast convergence time, which it achieves by pre-calculating backup routes. The primary, best path to a destination is called the **Successor**. A pre-qualified, loop-free backup path is called the **Feasible Successor**. If the Successor route fails, the router can immediately switch to the Feasible Successor without needing to recalculate the entire network topology, making it very efficient.

Why others are incorrect:

  • OSPF and IS-IS: These protocols build a map of the network and run an algorithm to find the single best path. If that path fails, they must run the algorithm again to find a new best path. They don't use the "feasible successor" terminology or concept.
  • BGP: As the routing protocol for the internet, BGP uses a complex set of path attributes (like AS-Path, Local Preference) to determine the best path between different autonomous systems. It does not use the successor/feasible successor model.

Question 2

A network administrator wants to configure a backup route in case the primary route fails. A dynamic routing protocol is not installed on the router. [cite_start]Which of the following routing features should the administrator choose to accomplish this task? [cite: 8, 9, 10]

A. [cite_start]Neighbor adjacency [cite: 11]

B. [cite_start]Link state flooding [cite: 12]

C. [cite_start]Administrative distance [cite: 13]

D. [cite_start]Hop count [cite: 14]

Show Answer & Explanation ▶

Correct Answer: C. Administrative distance

Why this is correct: Administrative Distance (AD) is a number (0-255) that a router uses to rank the trustworthiness of a routing source; a lower number is better. Since no dynamic protocol is used, the admin must use static routes. They can configure a "floating static route" by creating two static routes: one with the default AD (e.g., 1) and a backup route with a higher AD (e.g., 5). The router will install the primary route. If that route's interface goes down, the router removes it from the routing table and "floats" the backup route with the higher AD into place.

Why others are incorrect:

  • Neighbor adjacency & Link state flooding: These are functions of dynamic routing protocols like OSPF and EIGRP, which are not in use here.
  • Hop count: This is a metric used by protocols like RIP to measure the number of routers a packet must cross. It doesn't determine the trustworthiness or selection between two static routes.

Question 3

[cite_start]

Which of the following protocols is used to route traffic on the public internet? [cite: 15]

A. [cite_start]BGP [cite: 16]

B. [cite_start]OSPF [cite: 17]

C. [cite_start]EIGRP [cite: 18]

D. [cite_start]RIP [cite: 19]

Show Answer & Explanation ▶

Correct Answer: A. BGP

Why this is correct: The internet is a massive network of networks, where each network is an "Autonomous System" (AS). **BGP (Border Gateway Protocol)** is the only protocol designed to exchange routing information between these different ASes. It is the standardized Exterior Gateway Protocol (EGP) that makes global internet routing possible.

Why others are incorrect:

  • OSPF, EIGRP, and RIP: These are Interior Gateway Protocols (IGPs). They are designed to manage routing *within* a single autonomous system (like a single company's corporate network), not between them on the public internet.

Question 4

[cite_start]

Which of the following routing protocols uses an autonomous system number? [cite: 20]

A. [cite_start]IS-IS [cite: 21]

B. [cite_start]OSPF [cite: 22]

C. [cite_start]BGP [cite: 23]

D. [cite_start]EIGRP [cite: 24]

Show Answer & Explanation ▶

Correct Answer: C. BGP

Why this is correct: An **Autonomous System (AS)** is a collection of routers under a single administrative authority, identified by a unique **Autonomous System Number (ASN)**. **BGP** is the protocol that routes traffic *between* these different ASes on the internet. It fundamentally relies on ASNs to identify networks and make routing decisions.

Why others are incorrect:

  • EIGRP: While EIGRP configuration requires an "autonomous system" number, it's used locally to identify which routers belong to the same EIGRP domain and is not a globally unique number used for internet routing.
  • OSPF and IS-IS: These protocols operate within a single AS and do not use an ASN as part of their core routing logic. They use concepts like areas and process IDs for organization.

Question 5

[cite_start]

Which of the following protocols has a default administrative distance value of 90? [cite: 25]

A. [cite_start]RIP [cite: 26]

B. [cite_start]EIGRP [cite: 27]

C. [cite_start]OSPF [cite: 28]

D. [cite_start]BGP [cite: 29]

Show Answer & Explanation ▶

Correct Answer: B. EIGRP

Why this is correct: Each routing protocol is assigned a default **Administrative Distance (AD)** to indicate its trustworthiness (lower is better). For Cisco routers, the default AD for internal **EIGRP** routes is **90**. This makes it more preferred than OSPF (110) and RIP (120) but less preferred than a directly connected interface (0) or a static route (1).

Why others are incorrect (Common Cisco Defaults):

  • RIP: Default AD is 120.
  • OSPF: Default AD is 110.
  • BGP: Default AD is 20 for external BGP (eBGP) and 200 for internal BGP (iBGP).

Question 6

[cite_start]

In an environment with one router, which of the following will allow a network engineer to communicate between VLANs without purchasing additional hardware? [cite: 30]

A. [cite_start]Subinterfaces [cite: 31]

B. [cite_start]VXLAN [cite: 32]

C. [cite_start]Layer 3 switch [cite: 35]

D. [cite_start]VIP [cite: 36]

Show Answer & Explanation ▶

Correct Answer: A. Subinterfaces

Why this is correct: This scenario describes a classic "router-on-a-stick" configuration. **Subinterfaces** are logical, virtual interfaces created on a single physical router port. Each subinterface can be assigned to a different VLAN and given its own IP address to act as the default gateway for that VLAN. This allows a single physical router connection to route traffic between multiple VLANs.

Why others are incorrect:

  • VXLAN: This is an advanced tunneling protocol used to create virtual networks, often in large data centers. It's overly complex for this simple task.
  • Layer 3 switch: This is additional hardware that can perform routing, but the question specifies not purchasing additional hardware.
  • VIP (Virtual IP): This is a single IP address shared among multiple devices, typically used for redundancy (like in FHRP) or load balancing, not for inter-VLAN routing itself.

Question 7

[cite_start]

Which of the following allows an organization to map multiple internal devices to a single external-facing IP address? [cite: 37]

A. [cite_start]NAT [cite: 38]

B. [cite_start]BGP [cite: 39]

C. [cite_start]OSPF [cite: 40]

D. [cite_start]FHRP [cite: 41]

Show Answer & Explanation ▶

Correct Answer: A. NAT

Why this is correct: NAT (Network Address Translation) is a technology used on routers and firewalls to translate private, internal IP addresses (like 192.168.1.100) into one or more public, external IP addresses. This allows many devices on a private network to share a single public IP address to access the internet, conserving the limited supply of public IPv4 addresses.

Why others are incorrect:

  • BGP and OSPF: These are routing protocols that determine the best path for data, they don't translate addresses.
  • FHRP (First Hop Redundancy Protocol): This is a group of protocols (like HSRP, VRRP) that provide a redundant default gateway; it doesn't map multiple devices to one IP.

Question 8

[cite_start]

Which of the following should a company implement in order to share a single IP address among all the employees in the office? [cite: 42]

A. [cite_start]STP [cite: 43]

B. [cite_start]BGP [cite: 44]

C. [cite_start]PAT [cite: 45]

D. [cite_start]VXLAN [cite: 46]

Show Answer & Explanation ▶

Correct Answer: C. PAT

Why this is correct: PAT (Port Address Translation), also known as NAT Overload, is the most common form of NAT. It maps multiple private IP addresses to a single public IP address by also translating the source port number. For example, PC1's traffic might leave as `PublicIP:3001` and PC2's as `PublicIP:3002`. The router keeps a table of these mappings to direct return traffic to the correct internal device. This allows thousands of employees to share one public IP address.

Why others are incorrect:

  • STP (Spanning Tree Protocol): Prevents loops on Layer 2 switches.
  • BGP (Border Gateway Protocol): An internet routing protocol.
  • VXLAN (Virtual Extensible LAN): A network virtualization technology.

Question 9

[cite_start]

Which of the following technologies uses a VIP to provide gateway redundancy between two routers? [cite: 47]

A. [cite_start]LACP [cite: 48]

B. [cite_start]PAT [cite: 49]

C. [cite_start]FHRP [cite: 50]

D. [cite_start]BGP [cite: 51]

Show Answer & Explanation ▶

Correct Answer: C. FHRP

Why this is correct: **FHRP (First Hop Redundancy Protocol)** is a category of protocols (like HSRP, VRRP, GLBP) designed to prevent the default gateway from being a single point of failure. Two or more routers work together to present a single **VIP (Virtual IP)** address to the network clients. One router is "active" and handles the traffic, while the other(s) are on standby. If the active router fails, a standby router instantly takes over the VIP, ensuring clients always have a working gateway.

Why others are incorrect:

  • LACP (Link Aggregation Control Protocol): Bundles multiple physical ports into a single logical link for increased bandwidth and redundancy, but doesn't provide gateway redundancy.
  • PAT (Port Address Translation): Translates private IPs to a public IP.
  • BGP (Border Gateway Protocol): An internet routing protocol.

Question 10

A systems administrator installs multiple web servers and intends to use the First Hop Redundancy Protocol to control routing to the servers. [cite_start]Which of the following types of IP addresses should the administrator use to support this routing approach? [cite: 52, 53]

A. [cite_start]Virtual [cite: 54]

B. [cite_start]Gateway [cite: 55]

C. [cite_start]Static [cite: 56]

D. [cite_start]Dynamic [cite: 57]

Show Answer & Explanation ▶

Correct Answer: A. Virtual

Why this is correct: As explained in the previous question, **First Hop Redundancy Protocols (FHRP)** work by creating a **Virtual IP (VIP)** address. This VIP is not tied to any single physical device but is shared among the redundant routers or servers. Clients are configured to use this VIP as their destination, and the FHRP protocol ensures that one of the active devices will always respond to traffic sent to that virtual address.

Why others are incorrect:

  • Gateway, Static, Dynamic: These terms describe roles or assignment methods for IP addresses, but the specific type of address used by FHRP to create a redundant, shared presence is a virtual IP.

Question 11

[cite_start]

Which of the following most likely requires the use of subinterfaces? [cite: 58]

A. [cite_start]A router with only one available LAN port [cite: 59]

B. [cite_start]A firewall performing deep packet inspection [cite: 60]

C. [cite_start]A hub utilizing jumbo frames [cite: 61]

D. [cite_start]A switch using Spanning Tree Protocol [cite: 62]

Show Answer & Explanation ▶

Correct Answer: A. A router with only one available LAN port

Why this is correct: The primary use case for **subinterfaces** is to allow a single physical router interface to handle traffic for multiple VLANs. This is called a "router-on-a-stick" configuration. If a router needs to provide routing for several networks (VLANs) but only has one physical port to connect to the switch, subinterfaces are the solution. The physical port is connected to a trunk port on the switch, and each subinterface is configured to handle traffic for a specific VLAN tag.

Why others are incorrect:

  • Firewall, Hub, Switch using STP: None of these technologies inherently require subinterfaces. They are all related to different networking functions (security, Layer 1 connectivity, loop prevention).

Objective 2.2: Switching Technologies

Question 12

A network administrator is configuring a new switch and wants to connect two ports to the core switch to ensure redundancy. [cite_start]Which of the following configurations would meet this requirement? [cite: 64, 65]

A. [cite_start]Full duplex [cite: 66]

B. [cite_start]802.1Q tagging [cite: 71]

C. [cite_start]Native VLAN [cite: 72]

D. [cite_start]Link aggregation [cite: 73]

Show Answer & Explanation ▶

Correct Answer: D. Link aggregation

Why this is correct: Link Aggregation** (often using the protocol LACP, 802.3ad) is a technique that bundles multiple physical network connections into a single logical link. This provides two key benefits: increased total bandwidth (the sum of the links) and **redundancy**. If one of the physical links in the bundle fails, traffic will automatically continue to flow over the remaining active links, preventing a network outage.

Why others are incorrect:

  • Full duplex: Allows data to be sent and received simultaneously on a single link, but doesn't provide redundancy if the link itself fails.
  • 802.1Q tagging & Native VLAN: These are features used to manage VLANs on trunk links, they don't provide physical link redundancy.

Question 13

A network administrator received complaints of intermittent network connectivity issues. The administrator investigates and finds that the network design contains potential loop scenarios. [cite_start]Which of the following should the administrator do? [cite: 74, 75, 76]

A. [cite_start]Enable spanning tree. [cite: 77]

B. [cite_start]Configure port security. [cite: 78]

C. [cite_start]Change switch port speed limits. [cite: 79]

D. [cite_start]Enforce 802.1Q tagging. [cite: 80]

Show Answer & Explanation ▶

Correct Answer: A. Enable spanning tree.

Why this is correct: A switching loop occurs when there are multiple paths between two switches, causing broadcast frames to circulate endlessly, consuming all available bandwidth and crashing the network. The **Spanning Tree Protocol (STP)** is specifically designed to prevent this. It intelligently detects redundant links and blocks traffic on them, ensuring there is only one active path at any time. If the primary path fails, STP automatically unblocks the redundant path to maintain connectivity.

Why others are incorrect:

  • Port security: Limits which MAC addresses can connect to a port; it doesn't prevent loops.
  • Port speed limits: Manages bandwidth but doesn't solve the core issue of a broadcast storm from a loop.
  • 802.1Q tagging: Used for VLANs, not loop prevention.

Question 39

Which of the following are environmental factors that should be considered when installing equipment in a building? (Choose two.) [cite_start][cite: 246, 247]

A. [cite_start]Fire suppression system [cite: 248]

B. [cite_start]UPS location [cite: 249]

C. [cite_start]Humidity control [cite: 250]

D. [cite_start]Power load [cite: 251]

E. [cite_start]Floor construction type [cite: 252]

F. [cite_start]Proximity to nearest MDF [cite: 253]

Show Answer & Explanation ▶

Correct Answers: C. Humidity control and D. Power load

Why these are correct: Environmental factors relate to the physical conditions of the surrounding area that can affect equipment operation.

  • Humidity control: Too much humidity can cause condensation and corrosion on electronic components. Too little humidity increases the risk of electrostatic discharge (ESD), which can damage sensitive chips. Proper humidity control is critical.
  • Power load: This is the total amount of electrical power the equipment will draw. It's an essential environmental factor to consider to ensure the building's circuits can safely handle the demand without overloading and causing power failures or fires.

Why others are incorrect:

  • Fire suppression system & UPS location: These are crucial safety and infrastructure planning items, but they are systems you install *into* the environment, not properties *of* the environment itself like temperature or humidity.
  • Floor construction type: This relates to the physical support and weight capacity, a structural concern, not an environmental one.
  • Proximity to nearest MDF: This is a logistical and network design consideration related to cabling distance, not an environmental factor.
